{{ serverAPI || 'https://nanovault.io/api/node-api' }}
This server must be running the NanoVault Server application, or an API compliant with it.
Timestamps in the application are dependant on this server's access to them.
{{ serverNode || 'N/A' }}
This server must be exposing the Nano RPC to the API server used above.
Requests will still be sent to the API server, but this node will be used to communicate with the network.
{{ serverWS || 'wss://ws.nanovault.io/' }}
This server must be running the NanoVault Websocket application, or a WebSocket API compliant with it.
Note: Since this server only notifies about new transactions, most of the wallet functions fine without it.
